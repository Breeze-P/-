# turntable项目文档


### 项目部署

在线体验：https://turntable.web.cloudendpoint.cn

本地获取：

git clone git@github.com:Breeze-P/turntable_demo.git

npm install

npm run start

npm run build




### 项目实现功能

实现项目作业进阶版功能需求：

* 仿掘金UI九宫格抽奖
* 轻服务应用云函数、云数据库实现云端修改**礼品内容**、**抽奖爆率**、**计算得到抽奖结果**



### 系统设计

使用技术：react全家桶、node.js、轻服务

版本控制：git

设计过程：

* 创建云函数方便组件调用；
* 构建云数据库存储礼品信息；
* 逐层分解组件设计state；
* 实现游戏功能框架；
* CSS润色；
