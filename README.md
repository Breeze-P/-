# turntable项目文档


### 项目部署

在线体验：https://turntable.web.cloudendpoint.cn

本地获取：

git clone git@github.com:Breeze-P/turntable_demo.git

npm install

npm run start

npm run build


### 项目实现功能

基本实现项目作业基础功能需求：

* 仿掘金UI九宫格抽奖
* 手动更改奖品与奖品爆率
